 Brew Pod Coffee – Website Project
 Overview

Brew Pod Coffee is a modern, responsive coffee shop website designed to showcase products, categories, and provide a smooth user experience. The website allows users to browse different types of coffee, add items to a cart, and simulate a checkout process.

 Features
 Home Page
Hero section with featured coffee
Interactive featured products
Category previews
Customer reviews
 Categories Page
Clean card layout for coffee types
Easy navigation to product sections
 Products Page
Hot coffee, cold coffee, beans, and pastries
Add to cart functionality
Sort by price (low to high / high to low)
🛒 Cart System
Add items to cart
Cart stored using localStorage
Dropdown preview + full cart page
 Checkout System
Simulated checkout
Generates order numbers (1–99, resets after 99)
 Search Function
Users can search for items (e.g. latte, beans)
Redirects to relevant section
 About Page
Information about the coffee shop
Staff section
Customer-focused content
 Contact Page
Contact form
Business location included
 Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
LocalStorage (for cart system)
 Design

The website is inspired by modern coffee shop UI designs:

Coffee bean background with fade effect
Warm brown colour palette
Centered content layout
Clean and minimal user interface
